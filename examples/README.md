# Examples

Real-world teaser deployments for `imp.dev` resources.

Each folder contains:
- minimal manifests to demonstrate a use case
- a short README with prerequisites and apply/delete commands

## Teaser Packs

1. `secure-runner-pool` - Ephemeral CI runners that scale from webhook demand
2. `preview-env-pr` - Per-PR preview VM pattern
3. `warm-pool-fast-start` - Pre-warmed pool for low-latency assignment
4. `multi-tenant-isolation` - Team-level network and workload isolation
5. `edge-proxy-pop` - Node-pinned edge proxy placement
6. `batch-media-workers` - Elastic batch worker pool
7. `legacy-app-lift-and-shift` - Single-service VM workload pattern
8. `isolated-untrusted-builds` - Disposable sandboxed build execution
9. `compliance-segmented-workloads` - Sensitive workloads on dedicated nodes
10. `cost-optimized-night-jobs` - Scale-from-zero scheduled demand profile
